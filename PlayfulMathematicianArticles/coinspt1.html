Introduction

Today I will give you a puzzle on coin flipping. But the way we solved it almost seems like black magic.

The Puzzle

Lets think about how one bi-flip changes the number of coins. If there is no tails in those two coins the number of tails will increase by two. The number of heads will decrease by two. IF there is no heads the number of tails will decrease by two and the number of heads will increase by 2.If there is one head and one tail the numbers of heads will not change, the number of tails will not change. So the parity, (whether or not it is odd or even), of the number of tails stays the same. A composition of bi flips also preserves parity since after each bi-flip the parity doesn't change from one to the other, so overall there is no change. Now if the coin line is bi-flip solvable then there exist a composition of bi-flips which results in 0 tails, so the parity of tails equals the parity of 0. This means there is a even number of tails. So we have a condition that if false implies the coin line is not bi flip solvable.
